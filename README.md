# deuscheffler-preprocesamiento-cienciadatos

## Nombre del proyecto:
Preprocesamiento de Datos - Ciencia de Datos

## Objetivo:
El objetivo de este proyecto es aplicar técnicas de preprocesamiento de datos para preparar conjuntos de datos reales antes de su análisis o modelado.
Incluye pasos como limpieza, transformación, normalización, manejo de valores faltantes, y codificación de variables categóricas.

## Estructura básica de carpetas
preprocesamiento-cienciadatos/
│
├── data/                 # Conjuntos de datos sin procesar y procesados

│   ├── raw/              # Datos originales

│   └── processed/        # Datos limpios y transformados

│
├── .gitignore            # archivos excluidos para subir a GIT

│
└── README.md             # Descripción general del proyecto
